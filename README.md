Gallows-game

Gallows-game — это реализация классической словесной игры, где игроку необходимо отгадать скрытое слово, называя буквы. Игра фокусируется на логике, внимательности и расширении словарного запаса. Проект создан для демонстрации навыков разработки и структуры игровых приложений.

Описание игры

Игроку предлагается слово, скрытое набором пустых позиций.
Задача — по очереди угадывать буквы.

Если буква есть в слове — она открывается на соответствующих позициях.
Если буквы нет — фиксируется ошибка. Количество ошибок ограничено, после превышения лимита игра считается проигранной.

Игра завершается, когда:

игрок открыл все буквы,

или исчерпал лимит ошибок.

Особенности реализации

генерация случайных слов из заранее заданного списка;

обработка пользовательского ввода;

проверка корректности буквы;

отображение текущего состояния слова;

учёт количества ошибок;

система победы и поражения;

возможность начала новой игры.

Функционал

выбор случайного слова;

отображение прогресса — отгаданные и скрытые символы;

визуальное или текстовое отображение количества ошибок;

обработка повторных букв;

эндшкрин с результатом;

возможность перезапуска.
